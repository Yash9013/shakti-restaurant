# 🍽️ Shakti Restaurant

A premium, mobile-first website for **Shakti Restaurant** — a local favorite in Udhana, Surat, Gujarat.

## ✨ Features

- **Mobile-First Responsive Design** — Optimized for all devices
- **Conversion-Optimized** — Multiple CTAs, sticky mobile bar, one-tap calling
- **Premium UI/UX** — Smooth scroll animations, hover effects, modern typography
- **Trust Signals** — Customer reviews, ratings, Google Maps integration
- **Fast Loading** — Pure HTML/CSS/JS, no frameworks

## 🎨 Design

| Element | Details |
|---------|---------|
| Primary Color | Deep Warm Orange `#E67E22` |
| Secondary | Charcoal Black `#2C2C2C` |
| Background | Soft Cream `#FAF7F2` |
| Accent | Fresh Green `#27AE60` |
| Fonts | Poppins + Inter (Google Fonts) |

## 📋 Sections

1. **Hero** — Full-width with trust badge, CTAs, and stats
2. **About** — Story-driven with food imagery
3. **Menu** — Card layout with pricing (Daily Meals, Snacks, Beverages)
4. **Why Choose Us** — Icon-based feature highlights
5. **Gallery** — Masonry grid with hover overlays
6. **Reviews** — Customer testimonials and ratings
7. **Contact** — Address, hours, phone, embedded Google Maps
8. **Final CTA** — Strong closing conversion section
9. **Footer** — Quick links and branding

## 🚀 Getting Started

Simply open `index.html` in any modern browser. No build step required.
